Hi! My name is Radu Brehar. I am a software engineer with a focus on the web platform.

In the past 5 years I worked almost exclusively with JavaScript both on the front-end and the back-end. This means quite a lot of experience with everything in the web eco-system:

  • Node.js
  • NPM
  • browserify
  • HTML
  • CSS (including preprocessors)
  • SQL
  • ReactJS
  • jQuery
  • ExtJS
  • RequireJS
  • and others

Among other things, I have created ZippyUI, a JavaScript framework offering a complete set of modern UI widgets.

Lately my focus is modular development with npm and browserify. See my list of modules.

I favor functional programming.

I enjoy React, which makes UI development fun again.
